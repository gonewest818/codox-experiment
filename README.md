# codox-experiment

## Steps taken

1. `lein new codox-experiment` & add `lein-codox` plugin
2. configure codox output path to `gh-pages`
3. add `gh-pages` dir to `.gitignore`
4. `lein codox` to make sure docs go to the right place
5. commit and push
